绑定操作说明

  • 尾程渠道绑定马帮渠道，需点击【编辑】在弹窗中，马帮渠道编码下拉框中选择对应仓库在“仓库设置”中添加的马帮渠道。
  • 下拉默认选择“未绑定”，代表未与任何马帮渠道绑定，如已绑定过修改时选择“未绑定”则代表解绑
  • 马帮渠道仅允许与一个尾程渠道绑定，因此在编辑弹窗中点击【保存】需要判断马帮渠道是否与其他尾程渠道绑定，如绑定过提示“马帮渠道已绑定#尾程渠道名称#，请前往解绑后再操作”，其中#尾程渠道名称#为查询到的冲突尾程渠道名称。
